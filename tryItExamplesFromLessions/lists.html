Ordered list:

  1. First item
  2. Second item
  3. Third item
  4. etc...

You can change the sequence indicator by changing the attribute within the <ol> tag. For example:

<ol type="a"> tag.
  1. item a
  2. item b
  3. item c
  4. etc...
